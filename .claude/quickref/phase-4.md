# Phase 4: Strategy - Quick Reference

## Key Questions
1. Situation BEFORE this course?
2. Situation AFTER completing it?
3. One-sentence transformation promise?
4. Unique thesis or approach?

## Bloom's Taxonomy (Quick)
| Level | Verbs |
|-------|-------|
| Create | design, build, develop |
| Evaluate | assess, critique, justify |
| Analyze | compare, organize, deconstruct |
| Apply | implement, execute, use |
| Understand | explain, summarize, classify |
| Remember | list, recall, define |

## Kirkpatrick Levels (Quick)
| Level | Focus | Timing |
|-------|-------|--------|
| 1 | Reaction | Immediately |
| 2 | Learning | During/after |
| 3 | Behavior | 30-90 days |
| 4 | Results | 90+ days |
