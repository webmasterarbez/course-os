# Phase 7: Scripts - Quick Reference

## Script Formats by Type
| Content | Format |
|---------|--------|
| Video | Full script + [VISUAL], [B-ROLL] |
| Screencast | Voiceover + [ACTION], [ZOOM] |
| Slides | Speaker notes (SPEAK, TRANSITION) |
| Live | Facilitation guide |
| Audio | Audio script |

## Visual Cue Tags
```
[VISUAL: description]
[B-ROLL: description]
[GRAPHIC: description]
[ACTION: description]
[SCREENCAST: description]
[ZOOM: element]
[HIGHLIGHT: element]
```

## Video Script Structure
```markdown
## HOOK (0:00-0:30)
[VISUAL: ...]
Script text...

## OBJECTIVES (0:30-0:45)
[GRAPHIC: ...]
Script text...

## SECTION 1 (0:45-3:00)
...

## SUMMARY
...
```

## Voice Guidelines
- Conversational, not academic
- "You" not "learners"
- Short sentences
- Active voice
- ~150 words/minute

## Output
- `content/scripts/`

## Quality Gate
□ All lessons scripted
□ Format matches type
□ Voice consistent
□ Timing realistic
□ Visual cues included

## Command
`/course-scripts`
