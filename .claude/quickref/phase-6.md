# Phase 6: Content - Quick Reference

## Merrill's 5 Principles
| Principle | Question |
|-----------|----------|
| Problem | "Why does this matter?" |
| Activation | "What do I already know?" |
| Demonstration | "What does it look like?" |
| Application | "Can I do it?" |
| Integration | "How do I use it?" |
